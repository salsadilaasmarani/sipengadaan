# SiPengadaan — Website Project UAS Pengembangan Framework

Website profil project UAS pengembangan sistem purchasing berbasis framework custom.

## Struktur File

```
sipengadaan/
├── index.html        # Halaman utama (landing page)
├── docs.html         # Halaman dokumentasi framework
├── css/
│   ├── style.css     # Stylesheet utama
│   └── docs.css      # Stylesheet halaman docs
├── js/
│   ├── main.js       # JavaScript utama (animasi, terminal, nav)
│   └── docs.js       # JavaScript halaman docs (copy code, sidebar)
└── README.md
```

## Cara Pakai

1. **Buka langsung** — Double-click `index.html` di browser
2. **Live Server** — Pakai ekstensi Live Server di VS Code untuk hot reload

## Yang Perlu Dikustomisasi

### index.html
- Ganti `Nama Anggota 1–4` dengan nama asli anggota kelompok
- Ganti `NIM: 000000001–4` dengan NIM asli
- Ganti role sesuai pembagian tugas sebenarnya
- Ganti `Nama Dosen Pembimbing` dengan nama dosen
- Ganti nama universitas dan program studi
- Update status timeline sesuai progress pengerjaan

### Nama Project
- Default: **SiPengadaan** — bisa diganti di semua file HTML (cari: `SiPengadaan`)

## Halaman

| Halaman | File | Isi |
|---------|------|-----|
| Home | `index.html` | Hero, fitur, tech stack, timeline, tim |
| Dokumentasi | `docs.html` | Instalasi, struktur, routing, modul, database |

## Fitur Website

- ✅ Terminal typewriter animation di hero
- ✅ Animate on scroll (fade in)
- ✅ Responsive mobile
- ✅ Sticky navbar dengan scroll effect
- ✅ Timeline progress pengembangan
- ✅ Dokumentasi dengan sidebar + copy code button
